# Visited-Countries-Tracker
### PT
Este projeto tem o objetivo de dar track aos países que já visitei e quais desejo visitar. Ele funciona quase automaticamente, simplesmente posso editar o ficheiro htmlgen.py aqui pelo github ele já corre automaticamente e dá update ao mapa do github pages.
### ENG
This project has the objective to track which countries I visited and which I want to visit. He works almost alone, I only need to add the name of the country on htmlgen.py hereon github and he will run it and update the github pages alone.
